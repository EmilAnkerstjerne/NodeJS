JavaScript is a very lightweight programming language with a fairly easy syntax. JavaScript is a interpreted languange, not a compiled language. For a long time JavaScript only ran in browsers, until NodeJs came. NodeJs made it possible to have a JavaScript development environment. With NodeJs JS can now run on your system and it is possible to run servers with JS.
To run a program using node, the above command is used. 'node' plus the name of the JS file you want to run.
There is a great development tool for node, which is nodemon. Nodemon will run your program and restart everytime there is made a change, so you don't have to stop and run your code everytime. Nodemon is used similar to the node command.
To use nodemon you will have to install it using 'npm' - Node Packet Manager. Npm is used to install and use packages in NodeJs, which is an important part. To use npm to install packages your project will need to have a 'package.json' file, where everything will be listed.
