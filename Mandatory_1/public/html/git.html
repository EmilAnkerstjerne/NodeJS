The 'git clone' command will clone a repository. Tip: this is and easier way to create a git folder. Create repository on github and clone it instead of using 'git init'.
To start pushing to your github repository you will need to add the files to push with the 'git add' command. You can add folders, files or everything that has been changed with the keyword '-A' instead of the filename.
Next step is to commit your changes with 'git commit' the '-m' is to add a commit message, which is required. The commit message will come as the next argument.
Last step will be the 'git push' command. Everything committed will be pushed to your github repository.
